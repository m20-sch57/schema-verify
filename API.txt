bool receive_schema(user, task, file_name){
	- пользователь выбирает файл и нажимает кнопку "отправить"
	- пользователь user отправляем файл file_name по задаче task
	- программа возвращает результат лексической проверки
}

bool give_result(user, task, schema_number){
	- пользователь выбирает зачачу и вводит номер своей посылки
	- программа возвращает вердикты проверки на тестах
}
